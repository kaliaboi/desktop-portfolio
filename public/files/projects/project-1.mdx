# Desktop Portfolio

A retro-inspired portfolio website with a desktop OS interface.

## Overview
This project recreates the classic desktop experience in the browser, complete with draggable windows, icons, and file management.

## Tech Stack
- React & TypeScript
- Framer Motion for animations
- Tailwind CSS for styling
- MDX for content management

## Features
- Draggable and resizable windows
- Desktop icons with file associations
- Context menu support
- Mobile-responsive design
- Sound effects and haptic feedback
